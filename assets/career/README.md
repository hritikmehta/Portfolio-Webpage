# Career Images

Use this structure for production and future updates:

- `assets/career/2018-2023/01.webp` ... `04.webp`
- `assets/career/2023-present/01.webp` ... `04.webp`
- `assets/career/next/01.webp` ... `04.webp`

Recommended format:

- `webp` (preferred) or `jpg`
- 16:10 or 4:3 aspect ratio
- Suggested size: `1600x1000`
- Keep filenames zero-padded (`01`, `02`, `03`, `04`) for stable ordering.
